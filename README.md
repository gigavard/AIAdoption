# AIAdoption - Progetti di Studio

Repository contenente una raccolta di **progetti educativi** focalizzati su **AI Adoption** e applicazioni pratiche con tecnologie moderne.

## 📁 Progetti Contenuti

### 1. **FastAPIMagasine**
Progetto didattico che implementa un'**API REST CRUD** per la gestione di un magazzino con **FastAPI**.

- **Tecnologie**: FastAPI, Uvicorn, Pydantic
- **Scopo**: Imparare REST API, gestione dati in memoria, validazione dati
- **Cartella**: [`FastAPIMagasine/`](FastAPIMagasine/)
- **Avvio**: Vedi [`FastAPIMagasine/README.md`](FastAPIMagasine/README.md)

---

## 🚀 Come Usare

Ogni progetto è **auto-contenuto** con le proprie dipendenze:

```bash
cd <nome-progetto>
python -m venv venv
source venv/bin/activate  # su Windows: venv\Scripts\activate
pip install -r requirements.txt
```

---

## 📚 Struttura

Ogni progetto segue questa struttura standard:

```
<progetto>/
├── src/              # Codice sorgente
├── tests/            # Test unitari
├── docs/             # Documentazione specifica
├── requirements.txt  # Dipendenze del progetto
└── README.md         # Istruzioni del progetto
```
